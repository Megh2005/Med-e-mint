## Core Features:

- Doctor Finder: Allows users to input symptoms or diseases to find suitable doctors using a clean and intuitive search interface.
- Prescription Scanner: Users can scan and upload prescriptions using a drag-and-drop interface styled with neomorphism for easy digital storage and access. Generated by Gemini AI with a full-fledged architecture prompt which can decode the worst handwriting ever, and return a structured response based on constant data and prompts. Also supports 45 local languages using Google Translate via JQuery.
- AI Diet Coach: A generative AI tool that recommends meal plans for users after they enter their details such as height, weight, lifestyle, cuisine preferences, and demographics.
- Teleconsultation: Facilitates a secure video and audio connection with doctors for real-time consultations.
- Appointment Scheduling: Enables users to book and manage appointments with healthcare providers.
- Single Sign-On: Offers authentication via a centered "Sign in with Google" button to simplify user access, maintaining high security and eliminating the need for password management.
- Responsive Interface: Ensures the telemedicine website is fully responsive, offering a seamless user experience across all devices with consistently styled UI elements.

## Style Guidelines:

- Primary color: Bright blue (#4A90E2) to serve as the primary accent.
- Background color: Soft light gray (#F0F3F7) as the base for the neomorphism theme.
- Accent color: A lighter blue (#7DC4FF) to complement the primary, enhance call-to-action elements, and maintain a harmonious interface.
- Font: 'PT Serif' (serif) for a thin and modern look with high legibility.
- Use of modern, high-contrast icons to ensure clear visibility and quick recognition of functions. Use bright blue highlights.
- Maintain spacious alignment with proper contrast, rounded corners, and soft shadows to ensure a futuristic yet clearly visible medical design.
- Employ smooth hover animations on buttons, cards, and inputs, along with tactile press feedback and fluid micro-interactions using Framer Motion to enhance user engagement and satisfaction.
